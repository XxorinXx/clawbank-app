# Skills Registry (optional, attach only when needed)

Rule: Skills are OFF by default. Lead/Human enables per-story to avoid token bloat.

How to use in a story:

- In docs/STORY.md add a section:
  "Skills enabled: <skill ids/urls>"
- Only the named agent loads those skills for that story.
